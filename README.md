  # Prerequisites
  python3.9
  Django
  pip



  # Cloning
  In your terminal:

    $ git clone https://github.com/MARTIN-OMOLLO/INSTAGRAM.git
    $ cd GALLERY

  # Running the Application
  To run the application, in your terminal:

    $ python3.9 manage.py
  # Testing the Application
  To run the tests for the class file:

    $ python3.9
